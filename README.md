# sistema-gestao-eventos-culturais

##  Objetivo
Desenvolver um sistema para gerenciar eventos culturais, permitindo o cadastro de eventos, inscrições de usuários, controle de feedbacks e relatórios.

##  Tecnologias Utilizadas
- **Linguagem:** Python
- **Banco de Dados:** SQLite
- **Ferramentas de Modelagem:** Draw.io, Lucidchart, Figma
- **Controle de Versão:** GitHub

##  Estrutura do Projeto
- `/requisitos` → Documentação de requisitos e regras de negócio  
- `/diagramas` → Diagramas UML (casos de uso e classes)  
- `/docs` → Documentos adicionais e relatórios
