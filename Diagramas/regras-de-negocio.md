# Regras de Negócio

1. Um evento só pode ser **criado por um usuário administrador**.
2. Cada usuário pode **se inscrever em vários eventos**, mas **não pode duplicar inscrições**.
3. Feedbacks só podem ser **registrados por usuários que participaram do evento**.
4. Um evento deve ter **limite máximo de participantes**.
5. O sistema deve **bloquear cadastros incompletos** (campos obrigatórios).
