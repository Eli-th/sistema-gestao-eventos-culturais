# Requisitos Não Funcionais

1. O sistema deve ser **desenvolvido em Python** com interface simples e intuitiva.
2. O banco de dados deve ser **SQLite**.
3. O sistema deve garantir **armazenamento seguro de dados** dos usuários.
4. O sistema deve ter **tempo de resposta menor que 3 segundos** em cada ação.
5. O sistema deve ser **compatível com navegadores modernos** (Chrome, Firefox, Edge).
