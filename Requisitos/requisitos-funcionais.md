# Requisitos Funcionais

1. O sistema deve permitir o **cadastro de eventos culturais** (nome, data, local, descrição).
2. O sistema deve permitir o **cadastro de usuários** (nome, e-mail, telefone, senha).
3. O sistema deve permitir que os usuários **se inscrevam em eventos**.
4. O sistema deve permitir o **registro de feedbacks** sobre os eventos.
5. O sistema deve permitir a **visualização de relatórios** com número de inscritos por evento.
